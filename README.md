# cekeh-client
OpenGL client for cekeh game

<br />

## Requirements
 - cmake 
 - make 
 - g++ 

```bash
sudo apt-get install cmake make g++
```

## Running the application
```bash
# configure, build, and execute
./run

# force re-configure, re-build, and execute
./run -f # or
./run --force
```
